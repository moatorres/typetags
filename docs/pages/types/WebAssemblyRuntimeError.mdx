# TypeTags `.WebAssemblyRuntimeError`

### Overview

The `WebAssembly.RuntimeError` object is the error type that is thrown whenever WebAssembly specifies a [trap](https://webassembly.github.io/spec/core/exec/index.html#traps).

<Callout type="error" emoji="ï¸ðŸš«">
  Since error objects **do not** have unique type tags, a
  `WebAssembly.RuntimeError` type tag is the same as any other `Error` object.
  If you need to check if an error is an instance of a
  `WebAssembly.RuntimeError`, use `instanceof` instead.
</Callout>

### Usage

```js
import { TypeTags } from 'typetags'

try {
  throw new WebAssembly.RuntimeError('Hello', 'someFile', 10)
} catch (e) {
  console.log(TypeTags.get(e) === TypeTags.WebAssemblyLinkError)
  // â†’ true (oops!)

  console.log(e instanceof WebAssembly.LinkError)
  // â†’ false
}

console.log(TypeTags.WebAssemblyRuntimeError)
// â†’ [object Error]
```
