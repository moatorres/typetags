# TypeTags `.isEvalError`

### Usage

#### `TypeTags.isEvalError(value)`

<Callout type="error" emoji="ï¸ðŸš«">
  Since error objects **do not** have unique type tags, an `EvalError` type tag
  is the same as any other `Error` object. If you need to check if an error is
  an instance of an `EvalError`, use `instanceof` instead.
</Callout>

- Checks if `value` **is** or **has** a default `EvalError` type tag.

