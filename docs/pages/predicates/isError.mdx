# TypeTags `.isError`

### Usage

#### `TypeTags.isError(value)`

<Callout emoji="️⚠️">Error objects **do not** have unique type tags.</Callout>

- Checks if `value` **is** or **has** a default `Error` type tag.

```js
const { TypeTags } = require('typetags')

let error = new Error('Bam!')

TypeTags.isError(error)
// → true

TypeTags.isError(new TypeError('Oops!'))
// → true

TypeTags.isError(error.toString())
// → false
```

### Signature

```ts
isError(value: any): boolean
```
